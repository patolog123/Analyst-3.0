US-001: Фиксация нарушений безопасности через Telegram-бота
Как специалист по охране труда,
я хочу иметь возможность фиксировать нарушения безопасности через Telegram-бота,
чтобы автоматизировать процесс формирования предписаний и рекомендаций по устранению нарушений.

**Детализация:**
1. Типы фиксируемых нарушений:
   - Отсутствие СИЗ
   - Нарушение правил работы на высоте
   - Неисправность оборудования
   - Другие нарушения (с возможностью указать вручную)

2. Собираемые данные:
   - Тип нарушения (выбор из списка)
   - Описание нарушения
   - Фото/видео подтверждение
   - Местоположение (геометка)
   - Дата/время фиксации
   - Ответственное лицо (если известно)

3. Формируемые отчеты:
   - Автоматическое предписание с рекомендациями
   - Ежедневный/еженедельный отчет по нарушениям
   - Уведомления ответственным лицам

**Acceptance Criteria:**
1. Бот должен предоставлять интерактивное меню для выбора типа нарушения
2. Система должна сохранять все данные о нарушении с привязкой к дате/месту
3. По каждому нарушению автоматически генерируется предписание с рекомендациями
4. Отчеты должны быть доступны для скачивания в формате PDF
5. Уведомления должны отправляться в Telegram ответственных лиц
