Заголовок: Получение аналитического отчета по S&P 500 и отдельным акциям.
Акторы:Пользователь (Инвестор,трейдер)
Предусловие:Пользователь авторизован в системе 
AI-агент имеет доступ к актуальным рыночным данным
Ограничения:Исторические данные доступны только с 1995 года. Задержки в API могут повлиять на актуальность данных.
Триггер:Пользователь отправляет запрос на анализ
Основной сценарий
1.Система: Отображает интерфейс выбора параметров анализа
2.Пользователь: Выбирает S&P 500, период "1 месяц", фундаментальный анализ
3.Система: Запрашивает подтверждение выбора
4.Пользователь: подтверждает параметры проводимого анализа.
5.Система: Загружает рыночные данные (цены, индикаторы, фундаментальные показатели).
6.Система: Отображает статус обработки.
7.Система: Формирует отчет с торговыми рекомендациями; уровнем рисков прогноза, графиками цен.
8.Система: предлагает варианты экспорта (Exel,PDF,Mail).
9.Пользователь: подтверждает "Экспорт в PDF".
10.Система сохраняет отчет и уведомляет об успешном завершении.
Альтернативный сценарий:
1.Система: Отображает интерфейс выбора.
2.Пользователь: Выбирает только технический анализ.
3.Система: Предупреждает "фундаментальные факторы не будут учтены. Продолжить?"
4.Пользователь: Подтверждает ограниченный анализ.
5.Система: Выполняет и предоставляет ограниченный отчет.
Исключительный сценарий:
1.Система: Обнаруживает ошибку при подключении к API
2.Система: Предлагает - Повторить попытку; Использовать последние доступные данные.
3.Пользователь: Отменяет операцию.
4.Система: Завершает сценарий с ошибкой "данные недоступны"
Результат: Успех. PDf отчет сгенерирован и сохранен.
